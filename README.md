# Анализ тональности комментариев

##  Описание проекта

Данный проект представляет собой веб-приложение для анализа тональности пользовательских комментариев с использованием модели машинного обучения.

Программа позволяет:

- анализировать новый комментарий пользователя
- определять его класс
- выводить вероятности всех возможных классов (4 класса)
- сохранять результаты анализа в базу данных
- анализировать весь датасет комментариев
- получать статистику распределения классов
- просматривать примеры комментариев каждого типа

Приложение разработано с использованием **Streamlit**, а данные хранятся в базе **MySQL**.

---

# Классы комментариев

Модель классифицирует комментарии на 4 категории:

- **нормальный**
- **оскорбление**
- **угроза**
- **непристойность**

Для каждого комментария выводятся вероятности принадлежности к каждому классу.

---

#  Обучение модели

Модель машинного обучения была обучена в среде **Jupyter Notebook**.

В процессе обучения выполнялись следующие этапы:

1. загрузка и анализ датасета комментариев  
2. предобработка текста  
3. токенизация текста  
4. преобразование текста в последовательности  
5. padding последовательностей  
6. обучение нейронной сети  
7. оценка качества модели  

После обучения были сохранены:

- обученная модель  
- токенизатор текста  

Файлы сохраняются в папке `models`.

# База данных

Проект использует базу данных **MySQL**.

Название базы данных:

```
tone_comment
```

Таблица:

```
comments
```

Структура таблицы:

| поле | тип |
|-----|-----|
| id | INT AUTO_INCREMENT |
| comment | TEXT |
| prob_normal | FLOAT |
| prob_insult | FLOAT |
| prob_threat | FLOAT |
| prob_obscene | FLOAT |
| prediction | VARCHAR |

# Функциональность приложения

## 1. Анализ нового комментария

Пользователь может ввести комментарий и получить:

- предсказанный класс
- вероятности всех классов
- визуализацию результатов

После анализа комментарий сохраняется в базу данных.

---

## 2. Анализ всего датасета

Во второй вкладке можно:

- обработать весь датасет комментариев
- обновить вероятности классов
- посмотреть статистику распределения
- увидеть примеры комментариев по каждому классу

Также можно получать случайные подборки комментариев.
